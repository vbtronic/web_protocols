What is URL State?

Save any JavaScript object into the URL with LZ compression. Share links that restore full app state — no backend needed.

▶ Live Demo

Enter some data and click "Save to URL"

Usage

import { saveToUrl, loadFromUrl } from './packages/url-state/src/index.js';

// Save state to URL
saveToUrl({ name: 'Viktor', score: 100 });

// Load it back
const state = loadFromUrl();
console.log(state); // { name: 'Viktor', score: 100 }

What is Web Editor?

A visual CSS editor that lets users customize any website's appearance. Changes are saved to localStorage and persist across visits.

▶ Live Demo

Click the button below to activate the editor on this page. Then click "Select" in the panel and click any element to style it.

Usage

import { init } from './packages/web-editor/src/index.js';

// Activate the visual editor panel
init();

// Or just apply saved styles silently
import { applySaved } from './packages/web-editor/src/index.js';
applySaved();

What is Data Guard?

Validates data before saving to localStorage or sessionStorage. Define custom rules and never save bad data again.

▶ Live Demo

Enter data and click "Validate & Save"

Usage

import { rule, guardedSet } from './packages/data-guard/src/index.js';

const rules = [
  rule('email', 'email', 'nonempty'),
  rule('age', 'positive', 'int'),
  rule('url', 'url'),
];

const result = guardedSet('user', { email: 'a@b.c', age: 25, url: 'https://x.com' }, rules);
console.log(result); // { saved: true, errors: [] }
